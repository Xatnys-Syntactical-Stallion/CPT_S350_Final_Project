# CPT_S350_Final_Project: A Runtime Comparison of Radix

## Dependencies
Make sure that g++ and bash are available on your computer.
## Running the program
First, download tester.cpp and tester.sh into the same directory.  Run the following command said directory:
```
$ bash tester.sh 10000
```
The argument for tester.sh is the maximum number of microseconds that an individual test can take before the program decides to quit.
You should now see a `output.svg` in the same directory.  `output.svg` is the graph of recorded runtimes from the tests.  It can be opened using a web browser.
